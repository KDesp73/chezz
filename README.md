# chess

Chess Library and 2 player game written in C 

## Installation

### Installation Directories

Libraries: `/usr/lib`

Headers: `/usr/local/include/chess`

```bash
# 1. Clone
git clone https://github.com/KDesp73/chezz
cd chezz

# 2. Verify that all the tests are passing (optional)
make test

# 3. Install locally
make install
```
